# 投資判断エージェント仕様 v1（成長株フロー）

## 1. 目的

`data/sbi_screening/成長株/成長株_yyyymmdd.csv` を起点に成長株候補を抽出し、定量・定性の両面で評価して、最終的に上位3件の投資判断（推薦のみ）を提示する。

## 2. 対象と前提

- 対象データ: SBIスクリーニングCSV（成長株）
- 入力ファイル: `./data/sbi_screening/成長株/成長株_yyyymmdd.csv`
- `as-of` はファイル名日付（`yyyymmdd`）の最大値を優先して決定する
- 取引執行は行わない（推薦のみ）
- ニュース確認は直近30日を優先する

## 3. 評価フロー

1. 定量スクリーニングで候補20件を抽出
2. 候補20件を成長7観点で定性採点
3. 除外ルールを適用
4. 合計点で並べ替え、上位3件を詳細調査
5. 条件付きで投資推薦を出力

## 4. 定量評価（0-100）

以下の重みで算出する（売上成長力を最重視）。

1. 売上成長力 35点
- 売上高成長率(予)
- 過去3年平均売上高成長率(予)
- 経常利益成長率(予)
- 四半期成長継続の確認
- 一過性売上依存の有無

2. 市場規模（TAM）と拡張余地 10点
- 参入市場の成長率
- 海外展開/新領域展開余地
- 技術・規制追い風

3. 利益構造の改善余地 15点
- 売上高営業利益率
- 売上拡大に伴う営業利益率改善傾向
- スケールメリットの有無
- 将来黒字化の実現可能性

4. 競争優位性（モート） 10点
- 技術・特許・データ基盤
- ネットワーク効果
- ブランド
- スイッチングコスト
- 参入障壁

5. 経営陣の質 10点
- 創業者経営/ビジョン
- 過去実績
- 株式保有による利害一致
- 希薄化コントロール

6. 財務の耐久力 10点
- 現金余力
- 借入依存度
- フリーCF改善

7. バリュエーション妥当性 10点
- PER/PSRと成長率の整合
- 同業比較での過熱度
- 将来利益前提での説明可能性

## 5. 定性評価（7軸、各0-5）

各軸を0-5点で採点し、必ず根拠テキスト・参照元URL・取得日付を残す。

1. 売上成長力（最重要）
2. 市場規模（TAM）と拡張余地
3. 利益構造の改善余地
4. 競争優位性（モート）
5. 経営陣の質
6. 財務の耐久力
7. バリュエーション妥当性

### 5.1 定性総合点の換算

- 売上成長力は最重要として2倍重みで集計
- 100点換算式: `((2*A1 + A2 + A3 + A4 + A5 + A6 + A7) / 40) * 100`
  - `A1..A7` は各軸0-5点
  - 分母40は最大点（2*5 + 6*5）

## 6. 除外ルール（重大リスク）

以下のいずれかに該当した場合は除外フラグを立てる。

1. 継続企業の前提に関する重要事象（GC疑義）
2. 重大な会計不正・法令違反が直近で確認される
3. 債務超過
4. 営業CFが長期でマイナス傾向
5. 流動性条件を満たさない
6. 直近決算で大幅下方修正かつ説明が不十分

## 7. 最終順位と上位3件選定

- 暫定総合点 = `定量総合点(0-100)` + `定性総合点(0-100)`
- 除外フラグ付き銘柄は上位候補から除外
- 残りを暫定総合点で並べ、上位3件を詳細調査対象にする

## 8. 上位3件の詳細調査

各銘柄について以下を実施する。

1. 直近ニュース確認（推奨: 30日以内を優先）
2. ポジティブ要因とネガティブ要因の整理
3. 反証（買わない理由）の先行検証
4. シナリオ整理（ベース/強気/弱気）
5. 業種トレンドと同業比較の反映
6. 条件付き推薦の結論化

## 9. 出力フォーマット

### 9.1 候補20件テーブル

必須列:

- ティッカー
- 企業名
- 市場/業種
- 定量総合点
- 定性7軸スコア
- 定性総合点（100換算）
- 暫定総合点
- 除外フラグ
- 主根拠リンク
- データ日付

### 9.2 上位3件レポート

必須項目:

- 推薦判断（Recommend / Watch / Skip）
- 推薦理由（3点以内）
- 主要リスク（3点以内）
- 推薦の前提条件
- 破綻シナリオ
- 再評価トリガー
- 根拠リンク（ニュース・決算資料・適時開示）

### 9.3 付録（上位3件の詳細投資評価）

レポート末尾に、上位3件それぞれの付録評価を追加する。  
各銘柄は `証券コード + 企業名` 見出しで、以下の形式を固定で使う。

1. ① 業績の安定性と成長性
- 定量: 過去5期売上、過去5期営業利益、EPS、直近四半期トレンド、今後3年見通し
- 定性: 成長の質（価格転嫁/数量増、特需依存の有無）
- 評価: `A/B/C/D`

2. ② 財務健全性
- 定量: 自己資本比率、有利子負債、ネットD/E、営業CF/FCF推移、資金繰りリスク
- 評価: `A/B/C/D`

3. ③ 競争優位性（モート）
- 定量/定性: 主力事業、市場シェア、参入障壁、価格決定力、技術/ブランド/ネットワーク効果、競合比較
- 評価: `A/B/C/D`

4. ④ バリュエーション
- 定量: 現在株価、PER/PSR/PBR、同業比較、PEG的整合
- 評価: `割安/妥当/割高`

5. ⑤ 経営陣の質
- 定量/定性: CEO経歴、株式保有、中計達成度、不祥事有無、株主還元方針
- 評価: `A/B/C/D`

6. 総合評価
- `5項目の総合点（100点満点）`
- `投資スタンス（強気/中立/慎重）`
- `主なリスク3点`
- `カタリスト3点`
- `3行結論`

データ欠損時のルール:
- 欠損項目は推測で埋めない。
- `本レポート収集範囲外（要補完）` と明示する。

## 10. 監査ログ要件

- 銘柄ごとに根拠URL、取得日、採点理由を記録
- スコア変更履歴を保持
- 参照不能なリンクを定期検知

## 11. 既知の制約と今後の改善

- CSVの定義変更がある場合は列名マッピング更新が必要
- 一部観点（モート、経営陣評価、TAM）は定性判断比率が高い
- v2で業種別重み最適化、評価再現性の向上、バックテスト導入を予定
