# 投資判断エージェント仕様 v1

## 1. 目的

東証プライム上場銘柄から「出遅れ株候補」を抽出し、定量・定性の両面で評価して、最終的に上位3件の投資判断（推薦のみ）を提示する。

## 2. 対象と前提

- 対象市場: 東証プライム
- 基本流動性条件: 直近20営業日平均売買代金が5億円以上
- 補助条件: 時価総額が300億円以上
- 取引執行は行わない（推薦のみ）
- 無料優先モードではデータ鮮度制約を明示する

## 3. 評価フロー

1. 定量スクリーニングで候補20件を抽出
2. 候補20件に対して定性5軸で採点
3. 合計点で並べ替え、上位3件を詳細調査
4. 条件付きで投資推薦を出力

## 4. 定量評価（10指標）

以下の10指標を利用し、業種内で正規化して比較可能にする。

1. PBR
2. PER
3. ROE
4. EV/EBITDA
5. 配当利回り
6. 売上高成長率（3年CAGR）
7. 営業利益成長率（3年CAGR）
8. 営業CFマージン
9. 自己資本比率
10. ネットD/Eレシオ

### 4.1 スコアリング方針（v1）

- v1は重みなし（等重み）で運用
- 指標ごとにスコアを0-100へ変換
- 定量総合点は10指標の単純平均
- 定量総合点の上位20件を候補とする

## 5. 定性評価（5軸）

各軸を0-5点で採点し、必ず根拠テキスト・参照元URL・取得日付を残す。

1. 出遅れ要因の一時性
2. 成長ドライバーの実現確度
3. 経営品質・資本政策
4. 競争優位性
5. リスク耐性

### 5.1 スコアリング方針（v1）

- v1は重みなし（等重み）
- 定性総合点は5軸の単純合計（0-25点）

## 6. 除外ルール（重大リスク）

以下のいずれかに該当した場合は除外フラグを立てる。

1. 継続企業の前提に関する重要事象（GC疑義）
2. 重大な会計不正・法令違反が直近で確認される
3. 債務超過
4. 営業CFが長期でマイナス傾向
5. 流動性条件を満たさない
6. 直近決算で大幅下方修正かつ説明が不十分

## 7. 最終順位と上位3件選定

- 暫定総合点 = `定量総合点(0-100)` + `定性総合点を100点換算`
- 除外フラグ付き銘柄は上位候補から除外
- 残りを暫定総合点で並べ、上位3件を詳細調査対象にする

## 8. 上位3件の詳細調査

各銘柄について以下を実施する。

1. 直近ニュース確認（推奨: 30日以内を優先）
2. ポジティブ要因とネガティブ要因の整理
3. 反証（買わない理由）の先行検証
4. シナリオ整理（ベース/強気/弱気）
5. 条件付き推薦の結論化

## 9. 出力フォーマット

### 9.1 候補20件テーブル

必須列:

- ティッカー
- 企業名
- 業種
- 定量総合点
- 定性5軸スコア
- 定性総合点
- 暫定総合点
- 除外フラグ
- 主根拠リンク
- データ日付

### 9.2 上位3件レポート

必須項目:

- 推薦判断（Recommend / Watch / Skip）
- 推薦理由（3点以内）
- 主要リスク（3点以内）
- 推薦の前提条件
- 破綻シナリオ
- 再評価トリガー
- 根拠リンク（ニュース・決算資料）

## 10. 監査ログ要件

- 銘柄ごとに根拠URL、取得日、採点理由を記録
- スコア変更履歴を保持
- 参照不能なリンクを定期検知

## 11. 既知の制約と今後の改善

- v1は等重みであり最適化未実施
- データソース品質差の影響を受ける
- 無料優先モードは鮮度に制約がある（遅延・公開期間）
- v2で重み最適化、セクター別補正、バックテストを導入
